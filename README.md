## Objectives

- To develop a real-time Android messaging application.
- To allow users to register, login, and manage their profile.
- To enable friend search, friend request, and friend list management.
- To support text messages, image sharing, and voice messages.
- To provide basic chat customization such as nickname, profile picture, and chat theme.
- To prepare the application for testing and Google Play release.

## Things to Prepare

| Area | Preparation |
|---|---|
| Development Tool | Android Studio, Kotlin/Java or Flutter |
| Backend Service | Firebase, Supabase, or custom backend |
| Database | Firestore, Realtime Database, or PostgreSQL |
| Authentication | Email login, phone login, or Google login |
| Cloud Storage | Storage for images and voice messages |
| Chat Features | Text message, image message, voice message |
| Friend System | Add friend, accept request, remove friend, block user |
| UI/UX Design | Login page, chat list, chat room, profile page |
| Security | User authentication, file validation, data access control |
| Google Play Setup | Play Console account, signed app bundle, app icon, screenshots, privacy policy |

## Expected Outcome

| Output | Description |
|---|---|
| Android Chat App | A working mobile chat application |
| User Account System | Users can register, login, and manage their profile |
| Real-Time Chat | Users can send and receive messages instantly |
| Image Sharing | Users can attach and send pictures |
| Voice Message | Users can record and send voice messages |
| Friend Management | Users can search, add, accept, and remove friends |
| Chat Customization | Users can customize nickname, profile picture, or theme |
| Google Play Ready Build | App is prepared for testing and Play Store submission |

## Final Goal

The final goal is to develop a functional Android chat application that supports real-time communication, friend management, media sharing, and basic chat customization, with the app prepared for Google Play deployment.
